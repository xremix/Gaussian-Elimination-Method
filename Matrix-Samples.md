# List of matrix samples

## 3 x 3

```js
1 2 3 2
1 1 1 2
3 3 1 0

a = 5, b = -6, c = -3
```

```js
3 6 3 24
2 1 3 13
3 2 4 19

a = 1, b = 2, c = 3
```

```js
5 1 -2 3
1 3 2 5
-2 0 1 -1

a = 2, b = -1, c = 3
```

```
1 1 1 1
3 4 5 6
5 3 3 0

a = -1.5, b = 2, c = 0.5
```
